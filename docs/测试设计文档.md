## 一、概述

本文档定义 DeviceStudio 的测试策略、测试框架、测试用例和测试流程。

### 1.1 测试目标

| 目标 | 说明 |
|-----|------|
| **功能正确性** | 验证所有功能按需求正确实现 |
| **稳定性** | 长时间运行无崩溃、无内存泄漏 |
| **性能** | 满足实时性和响应时间要求 |
| **兼容性** | 跨平台（Windows/Linux）正常运行 |
| **可维护性** | 代码易于维护和扩展 |

### 2.3 CMake 测试配置

```cmake
# tests/CMakeLists.txt

enable_testing()

# 查找 Google Test
find_package(GTest REQUIRED)

# 收集单元测试源文件
file(GLOB_RECURSE UNIT_TEST_SOURCES "unit/*.cpp")

# 创建单元测试可执行文件
add_executable(unit_tests ${UNIT_TEST_SOURCES})

target_link_libraries(unit_tests PRIVATE
    GTest::gtest
    GTest::gtest_main
    GTest::gmock
    Qt6::Core
    Qt6::Test
    core
    communication
    utils
)

# 注册测试
gtest_discover_tests(unit_tests)

# 添加内存测试 (Linux)
if(UNIX AND NOT APPLE)
    add_custom_target(memcheck
        COMMAND valgrind --leak-check=full --error-exitcode=1 $<TARGET_FILE:unit_tests>
        DEPENDS unit_tests
    )
endif()
```

## 三、单元测试

### 3.1 单元测试原则

| 原则 | 说明 |
|-----|------|
| **FIRST** | Fast, Independent, Repeatable, Self-validating, Timely |
| **单一职责** | 每个测试只验证一个功能点 |
| **AAA 模式** | Arrange, Act, Assert |
| **边界测试** | 覆盖正常、边界、异常情况 |

### 3.2 设备管理单元测试

```cpp
// tests/unit/core/test_device_manager.cpp

#include <gtest/gtest.h>
#include <gmock/gmock.h>
#include "DeviceManager.h"
#include "MockDevice.h"

using ::testing::_;
using ::testing::Return;
using ::testing::NiceMock;

class DeviceManagerTest : public ::testing::Test
{
protected:
    void SetUp() override
    {
        manager = std::make_unique<DeviceManager>();
    }
    
    void TearDown() override
    {
        manager.reset();
    }
    
    std::unique_ptr<DeviceManager> manager;
};

// 测试：创建串口设备
TEST_F(DeviceManagerTest, CreateSerialDevice_Success)
{
    // Arrange
    QVariantMap config;
    config["port"] = "COM3";
    config["baud_rate"] = 115200;
    
    // Act
    IDevice* device = manager->createDevice(DeviceType::Serial, config);
    
    // Assert
    ASSERT_NE(device, nullptr);
    EXPECT_EQ(device->type(), DeviceType::Serial);
    EXPECT_EQ(device->state(), DeviceState::Disconnected);
    
    // Cleanup
    manager->destroyDevice(device->deviceId());
}

// 测试：创建设备后计数正确
TEST_F(DeviceManagerTest, CreateDevice_CountIncreases)
{
    // Arrange
    int initialCount = manager->deviceCount();
    QVariantMap config;
    
    // Act
    IDevice* device1 = manager->createDevice(DeviceType::Serial, config);
    IDevice* device2 = manager->createDevice(DeviceType::TcpClient, config);
    
    // Assert
    EXPECT_EQ(manager->deviceCount(), initialCount + 2);
    
    // Cleanup
    manager->destroyDevice(device1->deviceId());
    manager->destroyDevice(device2->deviceId());
}

// 测试：销毁不存在的设备
TEST_F(DeviceManagerTest, DestroyNonExistentDevice_ReturnsFalse)
{
    EXPECT_FALSE(manager->destroyDevice("non_existent_id"));
}

// 测试：设备查询
TEST_F(DeviceManagerTest, GetDeviceById_ReturnsCorrectDevice)
{
    // Arrange
    QVariantMap config;
    config["port"] = "COM3";
    IDevice* device = manager->createDevice(DeviceType::Serial, config);
    QString deviceId = device->deviceId();
    
    // Act
    IDevice* retrieved = manager->deviceById(deviceId);
    
    // Assert
    EXPECT_EQ(retrieved, device);
    
    // Cleanup
    manager->destroyDevice(deviceId);
}

// 测试：按类型过滤设备
TEST_F(DeviceManagerTest, FilterDevicesByType)
{
    // Arrange
    QVariantMap config;
    IDevice* serial1 = manager->createDevice(DeviceType::Serial, config);
    IDevice* serial2 = manager->createDevice(DeviceType::Serial, config);
    IDevice* tcp = manager->createDevice(DeviceType::TcpClient, config);
    
    // Act
    QList<IDevice*> serialDevices = manager->devicesByType(DeviceType::Serial);
    
    // Assert
    EXPECT_EQ(serialDevices.size(), 2);
    
    // Cleanup
    manager->destroyDevice(serial1->deviceId());
    manager->destroyDevice(serial2->deviceId());
    manager->destroyDevice(tcp->deviceId());
}
```

## 十、CI/CD 集成

### 10.1 GitHub Actions 测试配置

```yaml
# .github/workflows/test.yml

name: Tests

on: [push, pull_request]

jobs:
  test-windows:
    runs-on: windows-latest
    steps:
    - uses: actions/checkout@v4
    
    - name: Setup MSVC
      uses: ilammy/msvc-dev-cmd@v1
      
    - name: Install Qt
      uses: jurplel/install-qt-action@v3
      with:
        version: '6.6.0'
        arch: 'win64_msvc2019_64'
        
    - name: Configure
      run: cmake -B build -DBUILD_TESTS=ON
      
    - name: Build
      run: cmake --build build --config Release
      
    - name: Run Tests
      run: ctest --test-dir build -C Release --output-on-failure
      
  test-linux:
    runs-on: ubuntu-22.04
    steps:
    - uses: actions/checkout@v4
    
    - name: Install dependencies
      run: |
        sudo apt-get update
        sudo apt-get install -y qt6-base-dev cmake g++ lcov
        
    - name: Configure
      run: cmake -B build -DBUILD_TESTS=ON -DCMAKE_BUILD_TYPE=Debug
      
    - name: Build
      run: cmake --build build
      
    - name: Run Tests
      run: ctest --test-dir build --output-on-failure
      
    - name: Generate Coverage
      run: |
        cd build
        lcov --capture --directory . --output-file coverage.info
```
